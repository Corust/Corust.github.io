Css Notes 2: ID

text-align

specifies the horizontal alignment of text

text-decoration

Adds things such as underlines, overlines, strikethroughs, etc

font-weight

Bold or light text

font-family

specifies a d font for an element

<span>

Used for grouping and applying styles to inline elements

How to use it

Can be used on any element
When using an ID, the selector should start with "#"
